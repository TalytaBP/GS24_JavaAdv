# Descrição 
Criamos um sistema de cadastro onde o usuário tem a possibilidade fazer o controle de consumo de energia. Onde é possivel colocar o gasto mensal os aparelhos utilizados e quantificar as pessoas em sua moradia, para assim o sistema fazer recomendações e sobre o consulmo de energia.
